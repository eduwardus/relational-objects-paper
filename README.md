# In Search of the Object of Negation: A Relational Experiment

This repository contains the complete code and data for the paper:

> **"In Search of the Object of Negation: A Relational Experiment on the Emergence and Dissolution of Objects"**

The project constructs a relational dynamical model in which candidate objects emerge as observer-dependent stable regions, demonstrating that no invariant structure persists across observers.

## Requirements

- Python 3.8+
- Packages listed in `requirements.txt`
